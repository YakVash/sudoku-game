# Sudoku Game

An interactive, browser-based Sudoku game built entirely with vanilla JavaScript, HTML, and CSS — no frameworks, no dependencies.

## Features

- Puzzle generation with a valid unique solution
- Real-time input validation — highlights mistakes as you type
- Multiple difficulty levels
- Built-in timer to track solve time
- Clean, minimal UI that works on any screen size

## Tech Stack

- **HTML** — page structure
- **CSS** — layout and styling
- **JavaScript** — game logic, puzzle generation, and validation (vanilla, no libraries)

## Project Structure

```
sudoku-game/
├── index.html    # Landing / entry point
├── game.html     # Main game page
├── script.js     # All game logic
└── style.css     # Styling
```

## How to Run

No installation needed. Just open the file in your browser:

```bash
git clone https://github.com/YakVash/sudoku-game.git
cd sudoku-game
open index.html   # or double-click the file
```

Or open `index.html` directly in any modern browser.

## How to Play

1. Open the game and select a difficulty level
2. Fill in the empty cells — each row, column, and 3×3 box must contain digits 1–9 with no repeats
3. Incorrect entries are highlighted in real time
4. Complete the board to win!
